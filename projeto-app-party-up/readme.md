# 🎓 Party Up

## 📌 Descrição

O **Party Up** é um aplicativo Android focado em **estudos em conjunto**, conectando estudantes em tempo real através de videochamadas. A proposta é transformar o estudo individual em uma experiência colaborativa, aumentando a motivação, o foco e a produtividade.

Diferente de apps de chamada comuns, o Party Up cria um ambiente pensado especificamente para **sessões de estudo**, permitindo que usuários encontrem parceiros e estudem juntos de forma simples e rápida.

---

## 💡 Problema que o app resolve

Muitos estudantes enfrentam dificuldades como:

* Falta de motivação ao estudar sozinho
* Distrações frequentes
* Dificuldade em manter consistência

O Party Up resolve isso ao:

* Conectar estudantes automaticamente
* Criar um senso de compromisso
* Simular um ambiente de estudo em grupo

---

* **Android (Kotlin/Java)**

* **GitHub** (versionamento)

## 📱 Funcionalidades

* 🔗 Entrar em uma sala de estudo automaticamente
* 🎥 Videochamada em tempo real com outro estudante
* 💬 Chat durante a sessão
* ⏱️ Sessões focadas de estudo (modo produtividade)
* 👥 Sistema de conexão em dupla (match de estudo)

## ⚙️ Instalação

Siga os passos abaixo para rodar o projeto:

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/party-up.git
```

2. Abra o projeto no Android Studio

3. Aguarde o download das dependências

4. Execute o app em um emulador ou dispositivo físico

---

## ▶️ Como Usar

1. Abra o app
2. Clique em **"Encontrar parceiro"**
3. Aguarde a conexão automática
4. Inicie sua sessão de estudo por videochamada
5. Utilize o chat se necessário
